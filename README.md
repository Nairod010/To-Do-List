# Template webpack project

Template for future FE projects, kinda basic but this is the first version.

## What it has
- Webpack 5
- Configs for development and production
- CSS and images handling
- Live reload 

## Usage

1. Install dependencies:
```bash
./setup.sh
```
2. Run dev:
```bash
npm start
```
3. Build project:
```bash
npm run build

```
4. Deploy:
```bash
npm run deploy
```
